---
title: "Get top categories"
description: "Returns the top 5 expense categories by total amount for a given time period."
---

Returns the five categories with the highest total expense amounts, sorted in descending order, optionally filtered by month and/or year.

```bash
GET /api/stats/top-categories
```

## Request

### Query parameters

<ParamField query="month" type="number">
  Month to filter by. Must be an integer between 1 and 12.
</ParamField>

<ParamField query="year" type="number">
  Year to filter by. Must be within the range of years present in your transaction data.
</ParamField>

## Response

Returns an array of up to 5 category objects sorted by `value` descending. Returns an empty array (`[]`) if there are no expense transactions in the specified period.

<ResponseField name="name" type="string">
  Category name, capitalized (e.g., `"Food"`, `"Shopping"`).
</ResponseField>

<ResponseField name="value" type="number">
  Total amount spent in this category within the period.
</ResponseField>

<Info>Only expense transactions are included. Income transactions do not affect the results.</Info>

## Errors

| Status | Error message | Cause |
|--------|---------------|-------|
| 400 | `The month must be a number` | `month` query param is not a valid number |
| 404 | `Invalid month. Range: 1–12` | `month` is a number but outside the 1–12 range |
| 400 | `Invalid year. Available range: <min>–<max>` | `year` is outside the range of years in your data |

## Example

```bash
curl http://localhost:3000/api/stats/top-categories?month=3&year=2024
```

```json
[
  { "name": "Food", "value": 420 },
  { "name": "Health", "value": 185 },
  { "name": "Transport", "value": 210 },
  { "name": "Entertainment", "value": 150 },
  { "name": "Shopping", "value": 120 }
]
```
