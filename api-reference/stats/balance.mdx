---
title: "Get balance"
description: "Returns the total expense, income, and net balance for a given time period."
---

Returns the aggregate expense, income, and balance across all transactions, optionally filtered by month and/or year.

```bash
GET /api/stats/balance
```

## Request

### Query parameters

<ParamField query="month" type="number">
  Month to filter by. Must be an integer between 1 and 12.
</ParamField>

<ParamField query="year" type="number">
  Year to filter by. Must be within the range of years present in your transaction data.
</ParamField>

## Response

Returns a balance object, or `null` if no transactions match the filter.

<ResponseField name="expense" type="number">
  Total amount spent across all expense transactions in the period.
</ResponseField>

<ResponseField name="income" type="number">
  Total amount received across all income transactions in the period.
</ResponseField>

<ResponseField name="balance" type="number">
  Net balance calculated as `income - expense`.
</ResponseField>

<Info>If no transactions are found for the specified period, the response body is `null`.</Info>

## Errors

| Status | Error message | Cause |
|--------|---------------|-------|
| 400 | `The month must be a number` | `month` query param is not a valid number |
| 404 | `Invalid month. Range: 1–12` | `month` is a number but outside the 1–12 range |
| 400 | `Invalid year. Available range: <min>–<max>` | `year` is outside the range of years in your data |

## Example

```bash
curl http://localhost:3000/api/stats/balance?month=3&year=2024
```

```json
{
  "expense": 965,
  "income": 3500,
  "balance": 2535
}
```
