## What is Personal Finance API?

Personal Finance API is a REST API built with **Node.js and Express** that helps you manage your personal finances programmatically. Record income and expenses, filter transactions by date and category, and generate financial reports including balance summaries, category breakdowns, and month-over-month comparisons.

## API overview

The API exposes three resource groups:

| Resource | Base path | Description |
|---|---|---|
| Transactions | `/api/transactions` | Manage income and expense records |
| Statistics | `/api/stats` | Financial reports and summaries |
| Categories | `/api/categories` | Manage spending categories |

## Technology stack

- **Runtime**: Node.js (ES modules)
- **Framework**: Express 5
- **Validation**: Zod 4
- **Storage**: JSON file-based (in-memory during runtime)
- **CORS**: Configurable via environment variables

## Base URL

```
http://localhost:3000
```

All endpoints are prefixed with `/api`. For example:

```
GET http://localhost:3000/api/transactions
```
