## Prerequisites

- [Node.js](https://nodejs.org) v18 or higher
- [npm](https://npmjs.com) (included with Node.js)
- A terminal

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/wilker31vivas/personal-finance-api.git
    cd personal-finance-api
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```

    This installs Express 5, Zod, dotenv, and CORS.
  </Step>

  <Step title="Configure environment variables">
    Create a `.env` file in the root of the project:

    ```bash .env
    PORT=3000
    CORS_ORIGIN=http://localhost:5173
    ```

    | Variable | Required | Description |
    |---|---|---|
    | `PORT` | No | Port for the server. Defaults to `3000`. |
    | `CORS_ORIGIN` | Yes | Allowed origin(s). Use a comma-separated list for multiple origins. |

    <Warning>
      The `CORS_ORIGIN` variable is required. The server will crash on startup if it is not set.
    </Warning>
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    You should see:

    ```
    Servidor ejecutándose en puerto 3000
    ```

    The server restarts automatically when you save changes (uses `node --watch`).
  </Step>

  <Step title="Make your first request">
    Fetch all transactions:

    ```bash
    curl http://localhost:3000/api/transactions
    ```

    The response is a JSON array of transaction objects sorted by date (newest first):

    ```json
    [
      {
        "id": "a1f00101-aaaa-4a11-b111-000000000001",
        "type": "income",
        "amount": 3500,
        "category": "salary",
        "description": "Monthly salary payment",
        "date": "2024-01-05"
      },
      {
        "id": "a1f00102-aaaa-4a11-b111-000000000002",
        "type": "expense",
        "amount": 850,
        "category": "utilities",
        "description": "Rent payment",
        "date": "2024-01-10"
      }
    ]
    ```
  </Step>
</Steps>

## Create your first transaction

```bash
curl -X POST http://localhost:3000/api/transactions \
  -H "Content-Type: application/json" \
  -d '{
    "type": "expense",
    "amount": 45,
    "category": "food",
    "description": "Lunch",
    "date": "2024-11-15"
  }'
```

Response:

```json
{
  "id": "47047f7b-824b-4eb1-b41f-caaba7950d8e",
  "category": "food",
  "amount": 45,
  "date": "2024-11-15",
  "description": "Lunch",
  "type": "expense"
}
```

## Check your balance

```bash
curl "http://localhost:3000/api/stats/balance?month=11&year=2024"
```

```json
{
  "expense": 965.50,
  "income": 3500,
  "balance": 2534.50
}
```
